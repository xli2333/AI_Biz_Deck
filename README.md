# Strategy.AI Deck Builder

[中文版 README](README_CN.md)

**Strategy.AI** is an intelligent business presentation generation engine based on React and the Google Gemini 3.0 Pro model. It simulates the logical frameworks and visual styles of top consulting firms (McKinsey, BCG, Bain), automatically transforming your documents or brief ideas into professional strategic presentations.

![Project Banner](banner.png)

## New Features

### Custom Consulting Persona
No longer limited to the preset top three consulting firms.
- **Reverse Engineering**: Upload any PDF report or image you like, or enter a style description (e.g., "Cyberpunk minimalist style", "Professional Investment Banking style").
- **Style Extraction**: Gemini 3.0 Pro automatically analyzes and extracts its **Logic Core** and **Visual Identity**.
- **Instant Application**: The generated Custom Persona is directly applied to all subsequent slide generation, fully customizing everything from color schemes to narrative logic.

### 4K Upscale for NotebookLM
A visual enhancement workflow specifically designed for NotebookLM users.
- **Seamless Import**: Directly supports importing basic PDF slides generated by Google NotebookLM.
- **Intelligent Redrawing**: AI identifies the original content structure while preserving core information.
- **4K Enhancement**: Uses the `gemini-3-pro-image-preview` model to redraw originally blurry and simple charts into 4K ultra-high-definition, professional design-style business charts with one click.

## Core Features

This project is not just a "PPT generator"; it is an AI agent system that simulates the thinking process of a consulting advisor.

### 1. Context Intelligence
- **Multi-modal Input**: Supports uploading PDF, TXT, MD, DOC, DOCX documents, or directly entering your core Purpose.
- **Logical Deconstruction**: Built-in SCQA (Situation, Complication, Question, Answer) framework. The AI deconstructs your input like an analyst to extract key strategic information.

### 2. Consulting Personas
Built-in "Thinking and Visual" presets for three top consulting firms:
- **McKinsey Style**: Pyramid Principle, MECE principle, Classic Blue theme.
- **BCG Style**: Growth-Share Matrix, Insight-driven, Iconic Green theme.
- **Bain Style**: Answer First, Rigorous Data, Cardinal Red highlights.

### 3. Advanced Workflow
- **Ghost Deck**: Generates a text-only logical outline first, allowing you to confirm the narrative structure before visual generation.
- **Step-by-Step Construction**: Blueprint -> Render.
- **Real-time Control**: Supports **Pause/Resume** of the generation process for intervention at any time.

### 4. Refinement & Editing
- **Director's Cut**: Rewrite the entire outline logic with a single instruction.
- **Slide Refine**: Rewrite the logic for a specific single slide.
- **Visual Edit**: Modify image details using natural language.
- **Undo/History**: Complete version control.

### 5. Output
- **4K Upscaling**: Lossless upscaling to 4K Ultra HD.
- **PDF Export**: One-click synthesis of high-resolution PDF.

## Tech Stack

- **Frontend**: React 19, Vite, TypeScript
- **UI**: Tailwind CSS (v3.4), Lucide React
- **AI Core**: Google Gemini API (gemini-3-pro-preview / gemini-3-pro-image-preview)
- **Document**: `jspdf`, `pdfjs-dist`

## Getting Started

### Prerequisites
- Node.js (v18+)
- A valid **Google Gemini API Key** (Must start with `AIza`).

### Local Development

1.  **Clone Repository**
    ```bash
    git clone https://github.com/xli2333/AI_Biz_Deck.git
    cd AI_Biz_Deck
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Start Development Server**
    ```bash
    npm run dev
    ```

4.  **Access Application**
    Open browser at `http://localhost:3000`.

### Deployment

This project is fully adapted for **Vercel** deployment.
1.  Import Project in Vercel.
2.  **Important**: Set **Root Directory** to `AIDECK V1`.
3.  Click Deploy. No environment variable configuration needed; API Key is entered on the frontend.

## User Guide

1.  **Enter Key**: Input Gemini API Key on the landing page.
2.  **Provide Context**: Upload documents or enter purpose.
3.  **Select Style**:
    - Choose classic styles (McKinsey/BCG/Bain).
    - Or click **Custom** to upload a reference image/PDF for style mimicry.
    - Or click **Visual Engine** to import NotebookLM PDF for 4K redrawing.
4.  **Generate & Refine**: Generate Outline -> Build Deck -> Refine Details.
5.  **Export**: Click "Upscale (4K)" -> "Export PDF".
